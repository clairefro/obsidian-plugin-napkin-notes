# Napking Notes

Minimal Obsidian plugin for turning handwritten notes, sketches, and whiteboard photos into a simple, interactive image carousel.

## Features

- Filesystem upload or QR code upload from your phone
- Carousel viewing of images in reading mode
- Drag-and-drop reordering and deletion
- Optional descriptions for each image
- Responsive design for desktop and mobile

## Usage

1. Open a note and open the command palette (Ctrl+P or Cmd+P)
2. Run the command "Insert Napking Notes"
3. Upload images directly or scan the QR code to upload from your phone
4. Review, reorder, and annotate images
5. Insert into your note
6. View and navigate images in reading mode

## Markdown Format

Images are stored in a code block:

````markdown
```napkin-notes
[[attachments/your-image.jpg]]
Optional description
```
````

## Settings

- Custom upload folder
- QR upload server port range
- Enable/disable carousel in reading view
